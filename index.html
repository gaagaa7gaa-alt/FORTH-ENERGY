⚔️

Breaker専用コンボシステム

マイクラの常識を超えた、Breaker固有の超速連撃。3つのコンボルートを瞬時に切り替え、相手に選択肢を与えない。

  • combo1 — 踏み込み突破
  • combo2 — 電光石火の三角形
  • combo3 — 空中共闘
// 02
🎮

多彩なスキルと新技術

Jobごとに全く異なるスキルセット。マイクラでは有り得ないアクション体験。新しい動きを習得する新鮮さがある。

  • 職種ごとに独自のスキルツリー
  • 通常のマイクラにない動作・挙動
  • 練習と上達が実感できる設計
// 03
💥

環境破壊＆ド派手エフェクト

Long Gunnerを中心に、マップの一部を破壊するスキルが存在。フィールドを変えながら戦う戦略性と圧倒的な演出。

  • 一部マップを破壊可能なスキル
  • 雷光貫通・ソニックブームなど
  • 視覚と聴覚で最強を実感
// 04
🛡

被弾カウンター

金のハート（吸収属性）をリソースに変換。攻撃を受けることでコンボが起動する超攻撃的設計。守りから攻めへの即転換。

  • 守りが攻めに即転換
  • 敵の攻撃を敢えて受けて懐へ

Battle System

戦闘の仕組み

01

チームロール分担

2〜3チームが激突。各チーム3名、それぞれ異なるJobを担当することで補完的な戦略が生まれる。どのJobにも唯一無二の役割がある。

02

1回限りの命

死亡したら即観戦モードへ移行。復活なし。だから一瞬一瞬が濃密になる。試合開始時にランダムスポーンし、緊張感を最大化。

03

チーム専用ルーム

試合前、各チームには専用の準備ルームが割り当てられる。そこで戦略を立て、連携を固めてから戦場へ向かえ。

04

観客・実況席

試合を盛り上げる実況席と観客エリアを完備。その熱狂の渦をプレイヤー以外も体感できる。

05

訓練エリア

サーバー内に専用の訓練マップを用意。スキルの練習、コンボの習得、チームとの連携確認——本番前に完璧な準備を。

// Shield
20

プレイヤーは20ポイントのシールドを保有。攻撃されない限り自動回復し、満タン時に体力も回復する。

シールド管理が生存の鍵。
// Limit
▲

スキルと攻撃に必要なエネルギー。常に微量回復。Breakerはダメージを受けることで増加速度が跳ね上がる。

リミット不足はスキル封印と同義。
Rules & Map

ルール & マップ

01

DEATH = SPECTATE

死亡は即観戦移行。復活なし。一度の戦いに全てを懸けろ。

02

RANDOM SPAWN

試合開始時、指定エリアからランダムにテレポート。初動の判断力が勝負を左右する。

03

TEAM STRATEGY

専用ルームでチームの戦略会議を実施。連携なくして勝利なし。

04

RANKED SEASON

定期的に公式試合を開催し、順位を競う。常に頂点を目指せ。

05

TRAINING MODE

サーバー内に訓練エリアを完備。スキル・コンボ・連携を磨き上げろ。
